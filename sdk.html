Quickstart Guide

Three simple steps to integrate the CSGA MCP Platform into your application.

ICON

Installation

Install the SDK via npm for your preferred runtime environment.

npm install @csgaglobal/mcp-sdk
CFG

Initialize

Set up your connection to the CSGA platform with API credentials.

const mcp = new MCPClient({ apiKey: process.env.CSGA_API_KEY, endpoint: 'https://api.CSGA.global' });
ICON

Authenticate

Use API keys or OAuth 2.0 for secure authentication.

const token = await mcp.auth.login({ apiKey: YOUR_API_KEY });

Security

Authentication & Authorization

professional security for all API interactions and integrations.

API Keys

Simple API keys for development and testing. Scoped to specific resources and actions.

Authorization: Bearer YOUR_API_KEY

Best for: Development, server-to-server auth

OAuth 2.0

Industry-standard OAuth 2.0 for user-facing applications and third-party integrations.

grant_type=authorization_code &code=AUTH_CODE &client_id=YOUR_CLIENT_ID

Best for: User-facing apps, SaaS

JWT Tokens

JSON Web Tokens for stateless authentication and fine-grained access control.

Authorization: Bearer eyJhbGc...

Best for: Microservices, API gateways

mTLS Certificates

Mutual TLS for ultra-secure server-to-server communication in regulated environments.

client_cert.pem + client_key.pem

Best for: Classified, on-prem deployments

Security Best Practices

  • ICON Rotate API keys every 90 days
  • ICON Use environment variables for credentials
  • ICON Enable rate limiting and DDoS protection
  • ICON Audit all API access logs regularly
  • ICON Implement request signing for sensitive operations
