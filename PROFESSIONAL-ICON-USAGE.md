# MEOK AI Professional Icon System

## Available Icons

### Security & Compliance
- `icon-security` - Shield with lock (replaces 🔐)
- `icon-governance` - Balance scales (replaces ⚖️)

### Industry Sectors  
- `icon-healthcare` - Medical cross (replaces 🏥)
- `icon-ai-intelligence` - Neural network (replaces 🧠)

### Performance & Analytics
- `icon-analytics` - Chart bars (replaces 📊) 
- `icon-performance` - Lightning bolt (replaces ⚡)
- `icon-global` - Globe network (replaces 🌐)

### Actions & Status
- `icon-checkmark` - Professional checkmark (replaces ✓)
- `icon-consultation` - Person with chat (replaces corrupted chars)

## Usage Examples

### Basic Icon
```html
<span class="professional-icon icon-security professional-icon-md"></span>
```

### Icon with Text
```html
<div class="icon-text-container">
    <span class="professional-icon icon-governance professional-icon-md"></span>
    <span>Constitutional AI Governance</span>
</div>
```

### Button with Icon
```html
<a href="#" class="btn-professional">
    <span class="professional-icon icon-consultation"></span>
    Book Consultation
</a>
```

### Pillar/Feature Icon
```html
<div class="pillar-icon">
    <span class="professional-icon icon-security professional-icon-lg"></span>
</div>
```

## Size Classes
- `professional-icon-sm` - 16px (for inline text)
- `professional-icon-md` - 24px (default)
- `professional-icon-lg` - 32px (for cards/features)
- `professional-icon-xl` - 48px (for heroes/banners)

## Brand Colors
All icons use MEOK AI brand gradient colors:
- Primary: #3b82f6 to #1e40af (Blue)
- Success: #10b981 to #059669 (Green) 
- Warning: #f59e0b to #d97706 (Orange)
- Info: #06b6d4 to #0891b2 (Cyan)

## Accessibility
- All icons include appropriate titles
- Support for reduced motion preferences
- High contrast mode compatible
- Screen reader friendly markup
