# Travel Hospitality Ai MCP Server

> Part of the [MEOK AI Labs MCP Platform](https://meok-global.vercel.app/) — Cyber Security Global Alliance

Assess compliance for AI in travel and hospitality. Covers dynamic pricing, guest profiling, automated border control, loyalty manipulation, and accessibility.

## Installation

```bash
npx @smithery/cli install @meok-global/travel-hospitality-ai
```

## Configuration

Add to your MCP client config:

```json
{
  "mcpServers": {
    "travel-hospitality-ai": {
      "command": "npx",
      "args": ["-y", "@meok-global/travel-hospitality-ai"]
    }
  }
}
```

## Features

- AI governance compliance assessment
- Real-time policy validation
- CSOAI standards alignment
- Automated reporting
