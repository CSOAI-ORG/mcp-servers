# Ai Governance MCP Server

> Part of the [MEOK AI Labs MCP Platform](https://meok-global.vercel.app/) — Cyber Security Global Alliance

ISO 42001 compliance automation for AI systems with policy enforcement, audit trails, and risk management.

## Installation

```bash
npx @smithery/cli install @csgaglobal/ai-governance
```

## Configuration

Add to your MCP client config:

```json
{
  "mcpServers": {
    "ai-governance": {
      "command": "npx",
      "args": ["-y", "@csgaglobal/ai-governance"]
    }
  }
}
```

## Features

- AI governance compliance assessment
- Real-time policy validation
- CSOAI standards alignment
- Automated reporting
