# Energy Ai MCP Server

> Part of the [MEOK AI Labs MCP Platform](https://meok-global.vercel.app/) — Cyber Security Global Alliance

Assess compliance for AI in energy and utilities. Covers grid optimization, smart metering, demand response, renewable forecasting, and NERC/FERC regulations.

## Installation

```bash
npx @smithery/cli install @meok-global/energy-ai
```

## Configuration

Add to your MCP client config:

```json
{
  "mcpServers": {
    "energy-ai": {
      "command": "npx",
      "args": ["-y", "@meok-global/energy-ai"]
    }
  }
}
```

## Features

- AI governance compliance assessment
- Real-time policy validation
- CSOAI standards alignment
- Automated reporting
