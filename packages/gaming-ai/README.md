# Gaming Ai MCP Server

> Part of the [MEOK AI Labs MCP Platform](https://meok-global.vercel.app/) — Cyber Security Global Alliance

CSOAI Gaming & Entertainment AI Governance — loot box compliance, matchmaking AI fairness, content moderation, generative AI in gaming, COPPA, FTC, EU DSA, ESRB, and gambling classification

## Installation

```bash
npx @smithery/cli install @meok-global/gaming-ai
```

## Configuration

Add to your MCP client config:

```json
{
  "mcpServers": {
    "gaming-ai": {
      "command": "npx",
      "args": ["-y", "@meok-global/gaming-ai"]
    }
  }
}
```

## Features

- AI governance compliance assessment
- Real-time policy validation
- CSOAI standards alignment
- Automated reporting
